[quote, SixToSix]
____
:family: Six to Six ist eine digitale Unterstützung für pflegende Angehörige, die Tagesplanung,
Pflegedokumentation und Erinnerungen in einer einfachen, intuitiven App vereint.
____

== Hauptfunktionen
icon:users[] *Patientenverwaltung* (Name, Alter, Pflegegrad, Ansprechpartner)

icon:plus-square[] *Medikamentenplan* pro Patient

icon:calendar[] *Aufgaben- und Terminplanung* für Pflegekräfte

icon:file-text[] *Dokumentation* von Pflegeaktivitäten

icon:shield[] *Benutzerverwaltung* (z. B. Admin / Pflegekraft)
